## Quick start

```bash
# clone the repo
git clone https://github.com/jekabsmilbrets/exchange-rates.git

# change directory
cd exchange-rates

# install the repo with npm
npm install

# start the server
npm start

```
in your browser go to [http://localhost:4200](http://localhost:4200)
